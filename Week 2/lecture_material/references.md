## References

**You will be using references.cpp in `/src`**

In Java, everything passes by value. This means that a copy of the object is made when the object is passed as a parameter.
In C++, you can control how exactly things are passed using the reference operator `&`.

Passing something as a parameter to a function with the `&` operator will change how things work. Follow below to figure out exactly how.

#### 1. Compile and run `references.cpp`
   
#### 2. Notice the differences between the two arrays outputted to terminal
   
#### 3. Check over the code, see if you can spot what is the difference between the first and second functions in the `ChangeVector` class
   
#### 4. Taking this difference into account, consider how this affected the printed output in the `main` function
